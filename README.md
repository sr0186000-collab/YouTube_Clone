# 📺 YouTube Clone

A responsive YouTube-inspired frontend clone built using HTML5 and CSS3.

## 🚀 Project Overview

This project is a frontend practice project where I recreated the YouTube interface with a focus on responsive design, clean structure, and modern UI layouts.

The goal of this project was to improve my understanding of semantic HTML, CSS layouts, and responsive web design techniques.

## 🛠️ Tech Stack

- HTML5
- CSS3

## ✨ Features

- YouTube-inspired dark theme UI
- Responsive navigation bar
- Sidebar menu layout
- Search bar interface
- Category filter buttons
- Video card grid section
- Channel information layout
- Responsive design for different screen sizes
- Semantic HTML structure

## 📚 Learning Outcomes

- Improved HTML5 semantic structure
- Practiced CSS Flexbox and Grid
- Learned responsive web design concepts
- Improved UI recreation skills
- Better understanding of webpage layouts
